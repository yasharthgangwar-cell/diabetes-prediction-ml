# Diabetes Prediction using Logistic Regression

This beginner machine learning project predicts whether a person is likely to have diabetes using health parameters such as glucose level, BMI, blood pressure, and age.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Machine Learning Concepts Used
- Logistic Regression
- Train-Test Split
- Classification
- Model Evaluation

## Results
- Achieved ~75% model accuracy
- Implemented manual prediction for new patient data
