Diabetes Prediction using Machine Learning (Logistic Regression)
About the Project

This project is a simple machine learning model that predicts whether a person is likely to have diabetes based on health-related parameters such as glucose level, BMI, blood pressure, insulin, and age.

The goal is to understand how classification models can be used in healthcare prediction problems.

Objective

The objective of this project is to build a machine learning model that can predict diabetes risk based on patient medical data.

Dataset

The dataset contains medical records of patients with features such as:

Glucose level
Blood pressure
BMI
Age
Insulin
Skin thickness

The target variable indicates whether the patient has diabetes or not.
